Desktop App .NET

Impacto: De 200 procesos manuales a 1 click automático.
El Desafío de RRHH

El área de Recursos Humanos recibía un único archivo PDF con 200 recibos. Una persona debía separar manualmente cada hoja, identificar al empleado, renombrar el archivo y clasificarlo por periodo.

Solución Desarrollada:
  • Procesamiento de Documentos: Integración de librerías para la lectura y segmentación de PDFs de forma programática.
  • Extracción de Datos: Implementación de lógica para identificar automáticamente el legajo, nombre dentro del contenido de cada página.
  • Automatización de Archivos: Sistema de renombrado inteligente y organización en carpetas según el periodo correspondiente.

Optimización de Auditoría de Clientes

Desarrollé una solución para automatizar el análisis de encuestas telefónicas de satisfacción que anteriormente se encontraban dispersas en una base de datos MySQL.

Funcionalidades Técnicas:
  • Conectividad MySQL: Implementación de consultas eficientes para extraer respuestas de encuestas calificadas del 1 al 5.
  • Filtro de Rango Dinámico: Interfaz intuitiva para la selección de períodos de auditoría específicos (Desde/Hasta).
  • Análisis de Participación: El sistema calcula automáticamente el volumen total de llamadas frente al índice de respuesta efectiva.
  • Exportación Inteligente: Función de exportación a formato Excel (XLSX) lista para entrega directa al cliente final.
